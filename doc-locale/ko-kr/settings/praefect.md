[Gitaly Cluster (Praefect)](https://docs.gitlab.com/administration/gitaly/praefect/)는 리포지토리를 위한 내결함성 스토리지를 제공합니다. Gitaly를 위한 라우터 및 트랜잭션 관리자로 Praefect를 사용합니다.

## Gitaly Cluster (Praefect) 활성화 {#enable-gitaly-cluster-praefect}

기본적으로 Gitaly Cluster (Praefect)는 활성화되지 않습니다. Gitaly Cluster (Praefect) 활성화에 대한 정보는 Gitaly Cluster (Praefect) [설정 지침](https://docs.gitlab.com/administration/gitaly/praefect/configure/#setup-instructions)을 참조하세요.

## Gitaly Cluster (Praefect)가 활성화되었을 때 GitLab 업데이트 {#update-gitlab-when-gitaly-cluster-praefect-is-enabled}

Gitaly Cluster (Praefect)가 활성화된 상태에서 GitLab을 업데이트하는 방법에 대한 정보는 [특정 지침](https://docs.gitlab.com/update/zero_downtime/#upgrade-gitaly-cluster-praefect-nodes)을 참조하세요.
